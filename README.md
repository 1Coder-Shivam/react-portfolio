# Portfolio Website

A modern, responsive portfolio website built with React and Vite, featuring smooth animations and a clean dark theme.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Smooth Animations**: Scroll-triggered animations and interactive hover effects
- **Modern UI**: Dark theme with gradient accents and clean design
- **Fast Performance**: Built with Vite for optimal performance

## Tech Stack

- **React**: Frontend framework
- **Vite**: Build tool and development server
- **Tailwind CSS**: Utility-first CSS framework
- **Lucide React**: Modern icon library

## Sections

- Home / Hero
- Work Experience
- Projects
- Skills
- Contact / Get In Touch
